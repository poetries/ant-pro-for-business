<h2 align="center">基于Ant Design Pro改造业务系统 </h2>

### 系统特性

- 基于 `Ant Design` 体系精心设计
- 使用 `React/umi/dva/antd` 等前端前沿技术开发
- 针对不同屏幕大小设计
- 可配置的主题满足多样化的品牌诉求
- `Mock` 数据实用的本地数据调试方案

### 启动

```shell
git clone https://github.com/poetries/ant-pro-for-business.git

# install 
npm install

# start
npm start
```
